# UIDP Editor

一个基于 Excalidraw 的界面草图编辑器，用于生成 UIDP (User Interface Data Packet) 格式的几何数据文件，辅助 AI 进行界面开发。

## 功能特性

- 🎨 **手绘风格画布** - 基于 Excalidraw 的直观绘图体验
- 📐 **几何数据导出** - 将草图导出为结构化的 `.uidp` 格式
- 🏷️ **组件语义标记** - 为形状标记组件类型（button、input、label 等）
- 📱 **设备预设** - 支持移动端、平板、桌面等多种设备尺寸
- 📏 **自定义尺寸** - 支持输入自定义画板尺寸
- 🌐 **多语言支持** - 支持中文/英文界面切换
- 🤖 **AI 友好** - 简洁的数据格式，便于 AI 理解和处理
- 💻 **跨平台** - Windows 桌面应用，轻量快速

## 技术栈

- **Tauri v2** - Rust 驱动的桌面应用框架
- **React 19** - 用户界面
- **TypeScript** - 类型安全
- **Excalidraw** - 核心画布组件

## 快速开始

### 环境要求

- [Node.js](https://nodejs.org/) 18+
- [Rust](https://rustup.rs/) (Tauri 需要)
- Windows 10/11

### 安装依赖

```bash
cd uidp-editor
npm install
```

### 开发模式

```bash
npm run tauri dev
```

### 构建发布版本

```bash
npm run tauri build
```

构建完成后：
- 可执行文件：`src-tauri/target/release/uidp-editor.exe`
- 安装包：`src-tauri/target/release/bundle/`

## 使用指南

### 1. 选择设备预设

在顶部工具栏选择目标设备预设：
- **移动端**：375×667 (iPhone SE/8)
- **平板**：768×1024 (iPad 竖屏)
- **桌面 HD**：1280×720
- **桌面 FHD**：1920×1080
- **自定义**：手动输入宽度和高度

### 2. 绘制草图

使用 Excalidraw 工具绘制界面：

| 快捷键 | 工具 | 用途 |
|--------|------|------|
| R | 矩形 | 按钮、输入框、卡片 |
| O | 圆形/椭圆 | 头像、图标、单选框 |
| L | 线条 | 分割线、装饰线 |
| T | 文本 | 标签、标题、说明 |
| F | Frame | **画板** - 定义屏幕边界 |
| V | 选择 | 选择和编辑元素 |

### 3. 标记组件语义

1. 选中一个形状
2. 在顶部工具栏的「组件类型」下拉框选择类型：
   - `button` - 按钮
   - `input` - 输入框
   - `select` - 下拉选择
   - `checkbox` - 复选框
   - `radio` - 单选框
   - `textarea` - 多行文本
   - `switch` - 开关
   - `label` - 标签/说明文字
   - `image` - 图片占位
   - `container` - 容器/卡片

### 4. 导出数据

- **导出文件**：点击「导出 .uidp 文件」保存到本地
- **复制剪贴板**：点击「复制到剪贴板」直接复制内容

### 5. 切换语言

点击工具栏的 🌐 按钮可在中文/英文界面间切换。

## UIDP 文件格式 (v5.3)

`.uidp` 是一种极简的界面几何数据格式：

```uidp
# ═══════════════════════════════════════════════════════════
# UIDP PROTOCOL v5.3 - 界面几何数据格式
# ═══════════════════════════════════════════════════════════
#
# 【重要说明】
# - 所有形状的坐标都是相对于画板左上角的偏移量（相对坐标）
# - 渲染时需要使用等比缩放将相对坐标转换为实际像素位置
#
# 【渲染公式】
#   scale = min(实际窗口宽度 / 画板宽度, 实际窗口高度 / 画板高度)
#   实际X = 元素相对X * scale
#   实际Y = 元素相对Y * scale
# ═══════════════════════════════════════════════════════════

META:canvas=2560x1440 | unit=px | preset=desktop-hd | presetSize=1280x720

#0 | T:artboard | R:0,0,1280,720 | Z:0 | PRESET:desktop-hd
#1 | T:rect | R:80,90,200,50 | Z:1 | C:input
#2 | T:rect | R:80,160,200,50 | Z:1 | C:button
#3 | T:text | R:60,40,300,40 | Z:2 | TXT:用户名 | C:label
```

### 格式说明

- **META**：画布信息、设备预设和参考尺寸
- **#0**：画板（屏幕边界参考）
- **#N**：形状（类型、位置、层级、组件语义）
- **T**：类型（rect/circle/line/text/artboard）
- **R**：矩形区域（x,y,width,height）
- **Z**：层级（数字越大越在上层）
- **TXT**：文本内容（仅 text 类型）
- **C**：组件语义（可选）

## 画板说明

- 使用 **Frame 工具（F键）** 创建画板
- 面积最大的 Frame 会被自动识别为画板
- 画板定义屏幕边界，所有元素坐标相对于画板左上角
- 画板层级固定为 Z:0，始终在最底层

## 最佳实践

1. **先创建画板**：按 F 键创建 Frame 作为画板
2. **在画板内绘制**：所有元素应放在画板内部
3. **使用矩形表示组件**：按钮、输入框、卡片等用矩形表示
4. **添加文本标签**：使用文本工具添加标签和说明
5. **标记组件语义**：为关键组件添加语义标记，帮助 AI 理解
6. **设置正确预设**：导出前检查设备预设是否匹配目标平台
7. **使用自定义尺寸**：当预设不满足需求时，使用自定义尺寸输入
